# 脚本工具集

这是绘星痕个人脚本工具项目，包含多种实用的脚本工具，主要用于处理文件转换、音频处理、数据整理等任务。

## 项目结构

- 主程序入口：`main.py`
- 子工具目录：`tool***`（包含多个子工具）

## 功能说明

项目集成了多种实用工具，主要包括但不限于：
- 音频文件处理（格式转换、元数据提取、歌词提取等）
- 文件批量处理（移动、复制、解密、格式转换等）
- 数据整理与生成（Excel表格生成、M3U播放列表生成等）
- 其他辅助工具（GUI界面工具、自动化操作工具等）

部分常用子工具已制作GUI并集成到`main.py`中，方便直接使用；部分子工具尚未集成，可能存在潜在错误。

## 使用方法

1. 克隆或下载项目到本地
2. 安装必要的依赖（参见`requirements.txt`）
3. 从主程序入口启动：
   ```bash
   python main.py
   ```
4. 根据需要选择对应的工具功能进行使用

## 注意事项

1. **调试需求**：本项目为个人工具集，部分工具可能存在未完善之处，使用时可能需要自行调试
2. **环境依赖**：
   - 部分工具依赖`ffmpeg`，需要自行安装并配置环境变量
   - 其他依赖项请参考`requirements.txt`，使用`pip install -r requirements.txt`安装
3. **兼容性**：主要在Windows环境下开发测试，其他操作系统可能需要适当调整

## 子工具说明

各子工具位于`ScriptTools`目录下，主要功能如下：
- 音频处理：歌词提取、格式转换、音量标准化等
- 文件管理：批量移动、复制、删除重复文件等
- 数据处理：JSON文件修改、Excel表格生成等
- 其他工具：Spine文件处理、图片格式转换等

具体使用方法可查看各工具的代码注释。
